# Image API

A NestJS-based image upload API with:

- AWS S3 / MinIO storage
- Prisma + PostgreSQL
- Docker + docker-compose
- Unit and integration tests
- CI/CD with GitHub Actions

## Requirements

- Node.js **v24.11.1 (LTS)**
- npm **>= 10**
- Docker & Docker Compose (for local infra)

## Getting started

```bash
npm install
npm run start:dev
